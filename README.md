# Preparedness Commerce Project

Working repository for a modern, online-first preparedness / crisis-resilience commerce company for Germany.

## Current working hypothesis

Build a calm, highly trusted, design-forward preparedness brand for normal households and organizations: clear kits, official-guidance-aligned education, refill/reminder lifecycle, and modern digital distribution.

## Structure

- `research/market/` — competitor and market research by region.
- `research/brand/` — audience, positioning, messaging, naming, creative strategy.
- `research/product/` — kit architecture, product categories, bundle concepts, supply notes.
- `research/regulatory/` — legal/compliance sensitivities.
- `research/sources/` — raw source notes and URL inventories.
- `plans/` — step-by-step project plans and next actions.
- `docs/` — stable project context and decisions.
- `notes/` — quick working notes.
